Instructions:

  1. At the start of the game, you'll see a random collection value in the Crystals to Collect section.
  2. Collect crystals by clicking on the crystal kaleidoscope buttons displayed above.
  3. The value of each crystal is hidden until you click on it and then the value is added to the Number Collected.
  4. To win the game, match your Number Collected to the random Crystals to Collect number. But be careful! You lose the game if your total number of crystals collected is higher than the random number.
  5. Each time a new game starts, a new random collection value will appear and a new value will be assigned to each crystal.
