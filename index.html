Instructions:

  1. When the game starts, a random collection value will appear in the Crystals to Collect section.
  2. Collect crystals by clicking on the crystal kaleidoscopes displayed above.
  3. Each kaleidoscope has a value that stays hidden until you click on it and then the value is added to the Number Collected.
  4. To win the game, match your Number Collected to the random Crystals to Collect number. But be careful! You lose the game if your total number of crystals collected is higher than the random number.
  5. Each time a new game starts, a new random collection value will appear and a new crystal value will be assigned to each kaleidoscope.
